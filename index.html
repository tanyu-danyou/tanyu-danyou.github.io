新手GitHub Pages建站全流程｜0代码免费搭建个人网站

很多新手想建个人网站，怕花钱买服务器、怕写复杂代码，其实用 GitHub Pages 就能免费、稳定地搭建一个静态网站，全程跟着步骤来，10分钟搞定，后续还能申请 Google AdSense 挂广告赚点零花钱。

一、准备GitHub账号（0门槛注册）

首先，你需要一个免费的 GitHub 账号，它是我们搭建网站的基础。

  1. 打开 GitHub 官网：github.com
  2. 点击右上角的「Sign up」按钮，进入注册页面
  3. 输入你的邮箱地址、设置密码、创建用户名，注意：这个用户名非常重要，后面你的网站域名就是「用户名.github.io」
  4. 完成人机验证，点击「Create account」即可注册成功
⚠️ 避坑提醒：用户名一旦创建就不能修改，建议取一个和你网站主题相关、好记的名字，比如「tanyu-danyou」。

二、创建专属仓库（决定你的网站域名）

GitHub Pages 是通过仓库来部署网站的，你需要创建一个名字格式固定的仓库。

  1. 登录后，点击页面右上角的「+」号，选择「New repository」
  2. 在「Repository name」里，必须输入 你的用户名.github.io（比如你的情况就是：tanyu-danyou.github.io），一个字母都不能错
  3. 「Description」可以留空，也可以写你的网站介绍
  4. 勾选「Add a README file」，其他选项保持默认
  5. 点击「Create repository」，仓库就创建好了
💡 小贴士：只有名字格式为「用户名.github.io」的仓库，才会被 GitHub 自动识别为 Pages 网站，其他名字的仓库需要额外设置。

三、上传你的网页文件（index.html）

仓库创建好了，现在需要把你的网页文件上传进去，网站才能显示内容。

  1. 进入刚创建好的仓库，点击「Add file」→「Upload files」
  2. 把你写好的 index.html 文件拖到上传区域，或者点击「choose your files」选择文件
  3. 在下方「Commit changes」里，输入一个描述，比如「Initial commit」
  4. 点击「Commit changes」提交文件

这里有一个关键细节：仓库里必须有一个名为 index.html 的文件，它是网站的默认首页，没有它网站就无法正常显示。

四、部署并访问你的网站

文件提交后，GitHub 会自动帮你部署网站，这个过程一般需要2-5分钟。

  1. 部署完成后，打开浏览器，输入你的网站地址：https://你的用户名.github.io（比如你的就是：https://tanyu-danyou.github.io）
  2. 如果能看到你写的网页内容，就说明网站搭建成功了！
💡 总结：GitHub Pages 免费、稳定、不用维护服务器，是新手搭建个人博客、教程站的首选平台，也是申请 Google AdSense 最省事的方式。

Google AdSense注册全流程｜GitHub Pages网站审核避坑

GitHub Pages 网站做好后，下一步就是申请 Google AdSense，在网站上挂广告赚钱了！但很多新手会因为各种问题导致审核失败，这篇我把注册、验证、审核全流程的关键细节和避坑点讲清楚，帮你一次过审。

一、注册AdSense账号（用谷歌邮箱）

注册 AdSense 必须使用谷歌邮箱，没有的话需要先注册一个 Gmail。

  1. 打开 AdSense 官网：adsense.google.com
  2. 点击「开始使用」，用你的 Gmail 邮箱登录
  3. 在「您的网站」里，输入你的 GitHub Pages 网址，比如 https://tanyu-danyou.github.io
  4. 填写你的个人信息，包括国家、时区、收款人信息，注意信息要和后面税务信息里的一致
  5. 提交注册申请

二、网站所有权验证（过审的第一道门槛）

AdSense 必须确认这个网站是你本人所有的，才会继续审核你的申请，最常用的方法是「添加 AdSense 代码段」。

  1. 在 AdSense 后台，选择「验证您的网站」，方法选择「复制并粘贴代码段」
  2. 复制生成的验证代码，这段代码是唯一的，和你的账号绑定
  3. 打开你 GitHub 仓库里的 index.html 文件，把代码粘贴到 <head> 标签的最末尾
  4. 提交修改，等待网站更新生效（约2-5分钟）
  5. 回到 AdSense 后台，点击「验证」按钮，系统会检查你的网站是否包含这段代码
⚠️ 避坑提醒：
  • 代码必须放在 <head> 标签里，不能放在 <body> 里，否则验证会失败
  • 提交修改后，一定要等网站更新生效再点击验证，否则会提示找不到代码

三、提交审核并等待结果

所有权验证通过后，就可以提交网站审核了，这是决定你能不能通过的关键一步。

  1. 验证通过后，AdSense 后台会显示「您的网站正在审核中」
  2. 审核时间一般为 1-3 个工作日，慢的话可能需要一周，期间不要修改网站内容
  3. 审核结果会发送到你的 Gmail 邮箱，通过后你就能在后台看到「已批准」的状态

四、新手必看的审核避坑要点（通过率翻倍）

很多新手的申请被拒，都是因为踩了这些坑，一定要注意：

  1. 内容原创且有价值：网站里必须有你自己写的原创内容，比如教程、心得，不能是空页面，也不能直接复制别人的文章，AdSense 会识别抄袭内容直接拒绝
  2. 内容数量要足够：至少要有3-5篇完整的文章，每篇文章至少300字以上，有明确的主题和结构，不能只有几句话
  3. 网站无违规内容：不能有色情、暴力、赌博、政治敏感等内容，也不能放盗版、侵权的资源链接
  4. 网站能稳定访问：审核期间要保证 GitHub Pages 能正常打开，不能因为仓库设为私有或删除文件导致网站打不开
💡 我的经验：用 GitHub Pages 做的教程类网站，只要内容原创、结构清晰，通过率非常高，我自己就是一次通过的。

AdSense税务信息填写｜中国用户W-8BEN避坑（少扣20%税）

AdSense 审核通过后，必须填写税务信息，不然赚了钱也提不出来！中国用户填写 W-8BEN 表格，可以享受中美税收协定优惠，税率从30%降到10%，这步千万别填错，不然会被扣一大半的税。

一、税务信息入口在哪？

  1. 登录 AdSense 后台，点击左侧菜单的「付款」
  2. 选择「付款信息」，在页面下方找到「税务信息」模块
  3. 点击「添加税务信息」，开始填写

二、填写W-8BEN表格（中国用户专属）

进入税务信息填写页面后，系统会自动为中国用户匹配 W-8BEN 表格，这是专门给非美国纳税人用的声明，填写步骤如下：

  1. 身份信息：选择「个人」，姓名用拼音填写，必须和你收款银行卡上的拼音姓名完全一致，比如你的情况就是 YANG JINGCHI
  2. 地址信息：国家选择「中国」，地址必须用拼音或英文填写，不能写中文，否则会报错。比如你的地址可以写成：Liaojiazhen Cuicheng Community, Chongzhou City, Chengdu, Sichuan Province, China
  3. 税号信息：税号填写你的身份证号码即可
  4. 税务协定优惠：选择「是，我有资格享受更低的预扣税率」，国家选「中国」，收入类型勾选「服务或其他商业收入（如 AdSense）」，这样就能享受10%的优惠税率
⚠️ 避坑提醒：
  • 姓名拼音一定要和银行卡完全一致，顺序不能反，大小写不影响，但拼写不能错
  • 地址必须用英文/拼音，不能出现任何中文，否则无法提交
  • 一定要勾选税务协定优惠，不然会按30%的税率扣税，非常不划算

三、签名提交并等待生效

  1. 所有信息填写完成后，在「认证并签名」部分，输入你的拼音姓名签名，和前面填写的一致
  2. 勾选同意声明，点击「提交」
  3. 提交后，Google 会审核你的税务信息，一般1-2个工作日内生效，生效后你的收款就会按10%的税率扣税了
💡 总结：税务信息填写是 AdSense 收款的关键一步，只要信息和银行卡、身份证保持一致，地址用英文填写，一次就能通过审核，后续收款就没有问题了。
