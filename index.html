新手GitHub Pages建站全流程｜0代码免费搭建个人网站

很多新手想建个人网站，怕花钱、怕写代码，其实用GitHub Pages就能免费做！全程跟着步骤来，10分钟搞定，还能申请AdSense挂广告赚点零花钱。

一、准备GitHub账号

先注册GitHub账号，网址：github.com，用邮箱注册就行，免费。

二、创建专属仓库

1. 点右上角「+」→ New repository
2. 仓库名必须写：你的用户名.github.io（比如tanyu-danyou.github.io）
3. 勾选Add a README file，直接创建

三、上传网页文件

1. 新建index.html文件，写简单网页代码
2. 上传到仓库，提交更改
3. 等待2-5分钟，GitHub自动部署

四、访问你的网站

浏览器输入：https://用户名.github.io，就能看到自己的网站啦！

💡 小贴士：GitHub Pages免费、稳定、不用服务器，新手建站首选！

Google AdSense注册全流程｜GitHub Pages网站审核避坑

GitHub Pages网站做好后，就能申请Google AdSense挂广告赚钱了！但新手很容易踩坑导致审核失败，这篇把注册、验证、审核全流程讲清楚，帮你一次过审。

一、注册AdSense账号

1. 访问adsense.google.com，用谷歌邮箱注册
2. 输入你的GitHub Pages网址（xxx.github.io）
3. 填写个人信息，提交注册

二、网站所有权验证（关键）

1. 选择「AdSense代码段」验证
2. 复制代码，放到index.html的<head>标签里
3. 提交到GitHub，等待网站更新
4. 回到AdSense点击验证，通过后才能提交审核

三、审核避坑要点（必看）

1. 网站必须有原创内容（别抄别人的）
2. 至少3-5篇完整文章，不要只有一句话
3. 网站能正常访问，无违规内容
4. 审核时间1-3天，耐心等待

AdSense税务信息填写｜中国用户W-8BEN避坑（少扣20%税）

AdSense审核通过后，必须填写税务信息，不然赚了钱也提不出来！中国用户填W-8BEN表格，能享受中美税收协定优惠，税率从30%降到10%，这步千万别填错。

一、税务信息入口

AdSense后台→付款→付款信息→添加税务信息

二、填写要点（全是坑）

1. 身份选「个人」，国家选「中国」
2. 姓名用拼音（和银行卡一致，比如YANG JINGCHI）
3. 地址必须用拼音/英文，不能写中文（否则报错）
4. 税号填身份证号码
5. 收入类型选「服务或其他商业收入（AdSense）」

三、签名提交

最后输入拼音姓名签名，提交后1-2天生效，之后收款就只扣10%税了。

⚠️ 重要提醒：姓名拼音、地址英文千万别写错，否则审核失败！
